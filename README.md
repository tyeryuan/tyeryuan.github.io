# 个人知识库

这是一个基于 GitHub Pages 搭建的个人知识库网站，用于记录和分享学习笔记、技术心得和生活感悟。

## 项目结构

```
.
├── assets/          # 静态资源文件
│   └── styles.css   # 样式文件
├── posts/           # 文章内容
├── layouts/         # 页面模板
├── _includes/       # 可重用组件
└── index.html       # 主页
```

## 本地开发

1. 克隆仓库
2. 使用任意 HTTP 服务器在本地运行（如 Python 的 `http.server` 或 Node.js 的 `http-server`）

## 部署

本项目使用 GitHub Pages 自动部署，只需将更改推送到 main 分支即可。
